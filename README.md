# Creating an Angular Application

    <!-- Install the angular cli -->
    - npm install -g @angular/cli

    <!-- Install angular cli for specific version -->
    - npm install -g @angular/cli@19

    <!-- Check cli version -->
    - ng --version

    <!-- Create an app -->
    - ng new first-ng-app <!-- Optionally use --dry-run  (will not create any actual files)-->

    <!-- Create an app with some configuration -->
    - ng new first-ng-app --inline-style --inline-template <!-- Will create raw ts files for manual build (just for learning)-->

    <!-- Angular Devtools Extension -->

    - ng generate component header <!-- generates header component inside app/src -->

    - ng g c header <!-- short form -->
    
    - ng g c components/header <!-- generates header component inside app/src/components -->
